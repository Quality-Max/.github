## What We Build

QualityMax is an AI-native quality platform that covers four testing layers:

- **UI Testing** ‚ÄĒ AI crawls your web app, generates Playwright tests, self-heals when UI changes
- **API Testing** ‚ÄĒ Import OpenAPI specs or repos, generate endpoint tests with proper assertions
- **Performance Testing** ‚ÄĒ k6 load, stress, spike, soak tests with trend dashboards and regression detection
- **Security Scanning** ‚ÄĒ Integrated SAST with Semgrep, Bandit, and secret detection

Built by a QA engineer with 19 years of experience. Berlin-based.
